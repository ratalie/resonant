
RESONANT BONDS — FULL PACKAGE
Author: Natalie (N)

This ZIP contains all five tracks of the Resonant Bonds project,
ready to be opened in MuseScore and rendered in REAPER.

TRACKS
1. Healing Clarity — for You
2. Healing Containment — for Amabel
3. Healing Balance — for Danilo
4. Healing Depth — for Cristina
5. Healing Activation — for Jessica

All tracks:
- Instrumental
- Loopable
- 1–2 minutes
- Tuned to 432 Hz (via ReaPitch: -31.77 cents)

===============================
WORKFLOW — MUSESCORE
===============================

1. Open MuseScore
2. File → Open → select any .musicxml file
3. File → Save As → save as .mscz
4. (Optional) Add staff text:
   "Repeat freely. Let each note fully decay."
5. File → Export → MIDI (.mid)

===============================
WORKFLOW — REAPER
===============================

For EACH track:

1. Open REAPER
2. Create a NEW project (recommended)
3. Insert → Media File → select the .mid file
4. Set project tempo:
   Track 1: 60 BPM
   Track 2: 56 BPM
   Track 3: 88 BPM
   Track 4: 52 BPM
   Track 5: 96 BPM

5. FX chain on the track:
   - LABS (Soft Piano or Handpan)
   - ReaPitch → Shift cents: -31.77
   - Reverb (Valhalla Supermassive or similar)

6. Volume:
   - Track fader: -14 to -18 dB
   - Master peak below -1 dB

7. Loop:
   - Select full region
   - Press R
   - Enable Loop
   - Listen several minutes

===============================
RENDER (EXPORT AUDIO)
===============================

File → Render

- Source: Master mix
- Bounds: Time selection
- Format: WAV
- Sample rate: 44.1 kHz
- Bit depth: 24-bit
- No normalize, no limiter

Filename example:
N_Healing_Clarity_Track1.wav

===============================
FINAL NOTE
===============================

Do not overwork it.

This project is meant as a gift of care and presence.
